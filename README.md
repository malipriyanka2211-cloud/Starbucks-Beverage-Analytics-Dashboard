# Starbucks Beverage Analytics Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes Starbucks beverage nutritional information and global store presence. The dashboard helps users explore beverage categories, nutritional values, caffeine content, and Starbucks' worldwide expansion through interactive visualizations.

The objective of this project is to transform raw Starbucks beverage data into meaningful insights that support data-driven decision making.

# 🎯 Business Objective
Analyze nutritional values of Starbucks beverages.
Compare beverage categories based on calories, sugar, and caffeine.
Identify beverages with the highest caffeine content.
Visualize Starbucks global presence using geographical data.
Provide interactive filters for detailed analysis.

## 📊 Dashboard Features
1. KPI Cards

The dashboard displays important business metrics:

Total Beverage Count
Average Sugar
Average Calories
Average Caffeine
2. Average Calories by Beverage Category
Displays average calories across beverage categories.
Helps identify high-calorie and low-calorie beverage groups.

Visualization Used: Line Chart

3. Starbucks Global Presence
Shows Starbucks store locations worldwide.
Uses latitude and longitude coordinates for geographical visualization.

Visualization Used: Map Visual

4. Beverage Category Distribution
Displays the number of beverages available in each category.
Helps understand product diversity.

Visualization Used: Horizontal Bar Chart

5. Average Caffeine by Category
Compares average caffeine levels across beverage categories.

Visualization Used: Column Chart

6. Top 5 Highest Caffeine Beverages

Displays beverages with the highest caffeine along with:

Total Caffeine (mg)
Calcium (% DV)
Dietary Fibre (g)

Visualization Used: Table Visual

# Key Insights
Coffee beverages have the highest average caffeine content.
Frappuccino and Signature Espresso drinks contain higher calorie values.
Starbucks offers a wide variety of beverage categories.
The dashboard provides geographical insights into Starbucks' global store distribution.
