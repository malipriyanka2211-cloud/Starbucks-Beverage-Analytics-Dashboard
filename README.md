# Starbucks Beverage Analytics Dashboard

## Overview
This project presents an interactive dashboard analyzing Starbucks beverages.
It highlights nutritional insights, product mix, and global presence through
visual storytelling.

## Features
- **KPIs:** Average sugar, calories, caffeine across beverages
- **Comparative Charts:** Calories by category, caffeine distribution
- **Category Distribution:** Beverage mix visualization
- **Global Presence Map:** Starbucks footprint worldwide
- **Drill‑Down Table:** Top 5 highest caffeine beverages with nutrition details

## Tools & Skills
- Power BI for dashboard design
- SQL/Excel for data cleaning and modeling
- Data visualization and storytelling

## Key Insights
- Coffee and espresso dominate caffeine levels
- Smoothies and Frappuccino® categories are calorie‑heavy
- Balanced mix of beverage categories supports diverse customer preferences

## How to Use
1. Clone the repository
2. Open the `.pbix` file in Power BI Desktop
3. Explore interactive visuals and filters
